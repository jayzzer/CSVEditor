# CSVEditor

## Требования
1. .Net Framework
2. NuGet
3. MSBuild

## Сборка и запуск проекта
1. Установка зависимостей:
   ```bash
   nuget.exe restore CSVEditor.sln 
   ```
2. Сборка проекта:
   ```bash
   MSBuild.exe CSVEditor.sln
   ```
3. Запуск:
   ```bash
   ./CSVEditor/bin/Debug/CSVEditor.exe
   ```
